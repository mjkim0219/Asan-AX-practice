How it works

From prompt to PR
in minutes

Claude Code follows a tight loop: understand → plan → execute → verify, and keeps you in control at every step.

1

Install in seconds

Run npm install -g @anthropic-ai/claude-code and authenticate. Works on macOS, Linux, and Windows (WSL).

2

Describe your goal

Type a plain-English instruction — "refactor this module to use async/await" or "find the bug causing the 500 error." No special syntax required.

3

Claude explores & plans

The agent reads your code, checks git history, runs diagnostics, and proposes a plan before making any edits.

4

Execute & verify

Claude writes the code, runs tests, reads the output, and iterates until the task is complete — reporting evidence of success.
